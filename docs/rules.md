# Rules

Read /docs before generating code.

Stack:
React Native (web + mobile)
NestJS
PostgreSQL
Docker

Structure:
apps/api
apps/mobile
apps/web
packages/types

Backend:
Use NestJS modules.
Business logic in services.

Git:
- Use feature branches.
- Branch name: feature/<name>.
- Commit small logical changes.
- Use clear commit messages.
- When feature is finished, push branch.
- Merge to main when approved.
